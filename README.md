# user-feedback-form

### Project Description
An interactive feedback form built with HTML, CSS, and vanilla JavaScript. This project implements event-based programming, DOM manipulation, custom mouse-tracking tooltips, real-time character counters, and event delegation to capture and dynamically display user input.

### Features
* **Real-Time Character Counter:** Tracks user text input as they type.
* **Custom Tooltips:** Mouseover events reveal specific instructions dynamically.
* **Form Validation:** Blocks empty submissions and shows descriptive error messages.
* **Event Delegation:** Efficiently manages events from multiple fields using a single listener.
* **Event Propagation Control:** Uses `stopPropagation()` to isolate form interactions from background clicks.
